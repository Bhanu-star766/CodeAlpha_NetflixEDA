# 📊 Netflix EDA Project

## 📌 Overview
This project performs Exploratory Data Analysis on Netflix dataset to uncover trends, patterns, and insights.

## ❓ Questions Answered
- Movies vs TV Shows distribution
- Country-wise content production
- Content growth over time
- Ratings and genre analysis

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Key Insights
- Rapid content growth after 2015
- Movies dominate Netflix content
- USA leads in content production

## 🚀 Conclusion
The analysis highlights Netflix's expansion strategy and evolving content trends.
