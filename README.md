# 📊 Netflix EDA Project

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to uncover trends, patterns, and business insights.

## ❓ Business Questions
- What type of content dominates Netflix?
- Which countries produce the most content?
- How has content evolved over time?
- What ratings indicate audience preferences?
- Which genres are most common?

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Key Insights
- Rapid growth in content after 2015
- Movies dominate the platform
- USA leads content production
- Content targets mature audiences

## 🏁 Conclusion
The analysis highlights Netflix’s expansion strategy and evolving content trends, helping in data-driven decision making.

## 🔗 Project Files
- Netflix_EDA.ipynb (analysis notebook)
- netflix_titles.csv (dataset)
